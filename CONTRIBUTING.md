## 欢迎参军！

欢迎来到将军令项目！我们期待各路英才加入，共同打造中式编程语言。

## 如何贡献

### 报告问题
1. 在 Issues 中搜索是否已有类似问题
2. 使用模板提交新 Issue
3. 详细描述问题和重现步骤

### 贡献代码
1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/新特性`)
3. 提交更改 (`git commit -m '【类型】: 描述'`)
4. 推送到分支 (`git push origin feature/新特性`)
5. 创建 Pull Request
